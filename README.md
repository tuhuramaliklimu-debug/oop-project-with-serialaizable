# HOTEL MANAGEMENT SYSTEM
A comprehensive Hotel Management System built in Java with Swing GUI, providing complete solutions for room, customer, and reservation management with persistent data storage.

## FEATURES
### Room Management
* Add, edit, and delete rooms
* View room details: ID, type, status (FREE/BOOKED), capacity, price
* Automatic status updates based on reservations
* Color-coded status indicators

### Customer Management
* Complete CRUD operations for customers
* Store customer details: ID, name, email, phone, address
* Email format validation
* Duplicate ID prevention

### Reservation Management
* Create reservations with customer and room selection
* Auto-calculate total price
* Validate room capacity against guest count
* Status management: CONFIRMED, CANCELLED, COMPLETED
* Automatic room status synchronization

### Data Persistence
* Java Serialization for data storage
* Separate files for rooms, customers, reservations
* Auto-save on every operation
* Save prompt on application exit
